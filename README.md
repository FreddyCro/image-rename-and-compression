# Image Rename and Compress

## Requirements

- Node.js 14.18.1

## Usage

### 1. Install

```shell=
npm install
```

### 2. Initialize

```shell=
npm run init
```

### 3. Run

```shell=
npm start
```

1. rename: 只改檔案名稱。
2. compress: 壓縮 jpg / png 圖片檔案。
3. webp: 將 jpg / png 圖片轉成 webp 格式。
4. faker: 產生假圖片。
